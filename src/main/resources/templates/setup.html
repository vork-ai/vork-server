Vork is ready to use. Here’s what you can do:

  • Multi-provider AI chat — Gemini, OpenAI, and Ollama
  • Background task execution — AI agents running autonomously
  • Customisable agent templates — Save & reuse system prompts with scoped tools
  • SSH terminal integration — AI-assisted remote shell access
  • File upload & analysis — Attach files to any conversation
  • Tool authorisation — Fine-grained approval workflow for sensitive actions
  • Persistent session history — All conversations saved to MongoDB
  • Type generation — Auto-generate Java/TypeScript types from AI
  • Notification delivery — Email, SMS, and Telegram alerts via configurable providers
  • AI model discovery — Dynamic model lists fetched from provider APIs
